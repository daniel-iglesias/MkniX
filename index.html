MkniX

Nonlinear thermo-mechanical multibody systems simulation using FEM and meshfree methods

This project is maintained by daniel-iglesias

Overview

MkniX is a code design for the simulation of nonlinear flexible multibody systems, including thermal effects. It is organised as a library but, as it already has all necesary I/O functions, building a binary is trivial. Several examples of applications using the MkniX library are provided in the tests/ folder.
